# Fanuc CNC Indexer

A lightweight desktop application for indexing and searching FANUC CNC programs, built with Python and a graphical user interface.

## Features

- 🔍 Fast search through CNC program files
- 🗂️ Index and organize FANUC programs
- 🖥️ Simple and intuitive GUI
- ⚡ Lightweight — no internet connection required

## Requirements

- Python 3.x

## Usage

```bash
python app.py
```
